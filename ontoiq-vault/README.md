# Ontoiq Vault Structure

This directory contains the Obsidian vault for content management.

## Directory Structure

- **00-System/** - Obsidian templates and system guides
- **01-Raw-Content/** - Raw content from n8n (YouTube, RSS, etc.)
- **02-Extracts/** - AI-generated insights from OpenClaw
- **03-Drafts/** - Human-AI collaboration drafts
- **04-Published/** - Final published content
- **05-Templates/** - Content templates
- **06-Analytics/** - n8n analytics reports

## Sync

This vault is synced with Windows via Mutagen for Obsidian editing.
